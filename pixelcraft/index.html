07 / FAQ

Good questions, direct answers

Before you
hit register.

Is this the official VertechX website?+

No. This is an independent student portfolio concept created to demonstrate product design and frontend engineering. Official dates, fees and registrations must come from MVJCE.

Will the demo charge real money?+

No. The registration experience ends in a clearly marked sandbox simulation. A production version needs a verified Razorpay account and server-side payment signature validation.

Can this become a real event platform?+

Yes. The interface is designed around reusable event data and can be connected to Firebase Authentication, Firestore, Cloud Functions, Razorpay and an organizer dashboard.

What makes this resume-worthy?+

The value is not the visual layer alone. Document the problem, architecture, accessibility, performance, payment security model, testing and measurable improvements.
